### Vercel 이용한 배포
url : [사물인터넷TASK 채용과제 배포 url](https://amuse-assignment.vercel.app/)

#### 1. 이미지 경로문제
`src/assets/` 에 넣어둔 이미지는 배포환경에서 인식이 되지 않았다. 그래서 `public/images/`로 이동 후 절대경로로 이미지 호출

#### 2. msw worker동작
프로젝트 초반 개발환경에서는 msw호출을 막아뒀었는데, 이를 까먹고있다가 뒤늦게 알아차렸다. 배포환경에서도 msw를 사용해 mockData.json을 호출 해야하므로 모든 환경에서 worker를 사용해야했다.
```
// 배포 & 개발 환경에서 worker사용
worker.start().then(() => {
  createRoot(document.getElementById('root')!).render(
    <RecoilRoot>
      <BrowserRouter>
        <App />
      </BrowserRouter>
    </RecoilRoot>
  )
})
```
