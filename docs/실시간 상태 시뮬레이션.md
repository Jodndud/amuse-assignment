## 실시간 상태 시뮬레이션 구현
### 1. useEffect + setInterval 구성
- 컴포넌트가 마운트될 때만 실행되도록 의존성 배열은 []
- 언마운트 시 clearInterval로 정리
```
useEffect(() => {
  const interval = setInterval(() => {
    updateRandomDeviceStatus();
  }, 5000);

  return () => clearInterval(interval);
}, []);
```

### 2. Recoil 상태변경
- 조회는 필용없고 변경만 하므로 `useSetRecoilValue`사용
`const setDevices = useSetRecoilValue(devicesState);`

### 3. 랜덤 기기 개수 설정
- Recoil에 저장된 랜덤 기기를 선택하기 위해 `Math.random`과 `prev.length`를 이용한다
- 선택한 기기의 status를 `online` ↔ `offline` 변경한다
```
setDevices((prev) => {
  const randomIndex = Math.floor(Math.random() * prev.length);
  const target = prev[randomIndex];

  const newStatus = target.status === "online" ? "offline" : "online";
```

### 4. 적용
- 대쉬보드와 상세페이지 모두 적용하기 위해 `App.tsx`에서 `useRandomDeviceStatus()`을 실행시킨다.
